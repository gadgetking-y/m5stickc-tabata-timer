# M5StickC Tabata Timer

M5StickC と Speaker HAT を使用した、視認性が高くスムーズな動作のタバタ式トレーニングタイマーです。

## 特徴

- **スムーズな描画:** ダブルバッファリング（TFT_eSprite）の実装により、画面のちらつきを完全に解消。
- **高精度な音響フィードバック:** カウントダウン（3, 2, 1）と開始/終了音をビジュアルと完全に同期。さらに、全セット終了時には**盛大なファンファーレ**を演奏。
- **モダンなUI:**
  - 状態（WORK/REST/PREP）に応じたカラーバー表示。
  - 残り時間を示すスムーズなプログレスバー。
  - 残り3秒からのタイマー赤色強調。
  - **高視認性ラウンド表示:** トレーニング中でもひと目でわかる巨大なラウンド数表示（例: 8 / 8）。
- **スナッピーな遷移:** カウントが0になった瞬間に次フェーズへ即座に移行。最終セットのWORK終了時は休憩を挟まず、即座に終了画面とファンファーレへ遷移。

## ハードウェア要件

- **本体:** M5StickC (または M5StickC Plus/Plus2 ※コードの修正が必要な場合があります)
- **拡張:** M5Stack Speaker HAT

## 使い方

1.  **起動:** 電源を入れると「READY」画面が表示されます。
2.  **スタート:** 正面の大きなボタン (**Btn A**) を押すと 5秒間の準備（PREP）から開始します。
3.  **一時停止:** トレーニング中に **Btn A** を押すと一時停止します。
4.  **リセット:** 終了画面または一時停止中に右側面のボタン (**Btn B**) を押すと待機画面に戻ります。

## タイマー設定

コード内の定数を変更することで調整可能です。

```cpp
const int WORK_TIME = 20; // ワーク時間
const int REST_TIME = 10; // 休憩時間
const int PREP_TIME = 5;  // 準備時間
const int ROUNDS = 8;     // セット数
```

## インストール方法

1.  Arduino IDE または `arduino-cli` を用意します。
2.  `M5StickC` ライブラリをインストールします。
3.  `TabataTimer.ino` を開き、M5StickC に書き込みます。

---

### トラブルシューティング

**書き込み時にエラーが出る場合:**
- ポートが正しく選択されているか確認してください。
- `Connecting...` と表示されたら、本体の **M5ボタン（正面ボタン）を長押し** してブートモードに入れてください。

**音が出ない場合:**
- Speaker HAT が奥まで差し込まれているか確認してください。
- Speaker HAT の側面にあるボリュームダイヤルを確認してください。
